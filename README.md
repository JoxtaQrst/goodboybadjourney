# Good Boy, Bad Journey

*Good Boy, Bad Journey* is a 2D **platformer** game where you control a courageous dog on a mission to save its owner. Since his owner got kidnapped, the collar started to flicker, and so the dog can become invisible—but only when standing still. Not quite a the trinket, right ? As the player, you'll need to use this invisibility mechanic strategically navigate through levels and find his owner.

## Gameplay

- **Main Mechanic**
- **Levels**
- **Enemies & Obstacles**
- **Platforming**

## Features

- **Simple Controls**: Arrow keys (or WASD) to move, with the dog becoming invisible when standing still.
